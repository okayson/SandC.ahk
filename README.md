
# SandC

Converts key input while the space key is held down into Control + key.

## Install
```bash
$ git clone https://github.com/okayson/SandC.ahk.git
```

If you want to use as windows "exe", run the following.

```bash
$ compile.bat
```

If you want do add startup, run the following.

```bash
$ install.bat
```

If you want to remove from start app, run the following.

```bash
$ install.bat clean
```
